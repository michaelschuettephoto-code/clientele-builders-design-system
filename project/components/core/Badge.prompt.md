**Badge** ‚Äî compact mono-caps status pill mapped to the brand signal system.

```jsx
<Badge tone="healthy" dot>Healthy</Badge>
<Badge tone="leak" dot>Leak detected</Badge>
<Badge tone="scan" variant="solid">8-part diagnostic</Badge>
```

Tones: `neutral`, `scan` (teal), `leak` (amber), `healthy` (green), `brand` (navy). Variants: `soft` (tinted, default) / `solid` (filled). Add `dot` for a status indicator.
